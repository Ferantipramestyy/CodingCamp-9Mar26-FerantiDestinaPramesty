Project: To-Do List Life Dashboard

Tools:
- HTML
- CSS
- Vanilla JavaScript
- Local Storage

Features implemented:
- Greeting with time and date
- Pomodoro focus timer (25 minutes)
- To-do list with add, edit, delete, mark as done
- Quick links to favorite websites
- Dark mode
- Prevent duplicate tasks
